第04章 后端接口设计

辽宁师范大学 • 张大为@https://daweizh.github.io/jw2/

数据库中的数据需要通过后端逻辑组织向前端呈现，前端提交的数据需要通过后端逻辑向数据库中写入。后端业务逻辑通过RESTful风格的API形式向前端提供服务。前后端交互数据的格式采用JSON形式。

关于RESTful API设计规范请参考：

  1. 浅淡RESTful api设计规范，https://www.jianshu.com/p/9bb7e6c683a3
  2. RESTful API 设计指南，http://www.ruanyifeng.com/blog/2014/05/restful_api.html

关于JSON数据格式请参考：

  1. JSON官网，http://www.json.org/json-zh.html
  2. json.cn，https://www.json.cn/

以下针对AMC项目基本数据表的API进行初步设计。
